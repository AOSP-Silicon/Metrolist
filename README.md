# Metrolist

<img src="https://i.ibb.co/ykXB6rq/Ic-launcher.png" alt="Ic-launcher" border="0"></a>

YouTube Music client for Android

## Features

- Play songs from YT Music without ads
- Background playback
- Login support
- Library management
- Cache and download songs for offline playback
- Synchronized lyrics
- Skip silence
- Adjust tempo/pitch
- Material Ui
